# ProjectManager
CRUD operations on managing projects 

1. Read existing projects list, Read "ProjectDetails.json" from Files folder. This create a project under the folder with same Project name in Files Folder.
2. Update Project, takes action and update ProjectDetails.json (Missing to update any specific file only)
3. Delete Project, deletes item from ProjectDetails.json (as of now, just updating json and not deleting physical folder from Files Folder)
4. Download, this will create a zip folder with the same Project name in Files folder. (Missing feature: override existing same project name zip)
